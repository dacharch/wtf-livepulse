# 🔥 WTF LivePulse

A real-time gym analytics dashboard with live monitoring, anomaly detection, and simulation.

## 🚀 Features

- 📊 Real-time dashboard (WebSocket)
- 🏋️ Gym occupancy tracking
- 💰 Revenue analytics
- 🚨 Anomaly detection (capacity, zero check-ins, revenue drop)
- 🎮 Simulator for live events
- 📈 Analytics dashboard (charts)

## 🛠 Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL
- Realtime: WebSocket
- Deployment: Docker

## 🐳 Run Locally

docker compose up --build
